# 重庆邮电大学数学建模实验课程复习仓库

本仓库旨在帮助大家复习和巩固重庆邮电大学数学建模实验课及理论课的相关知识。仓库内包含实验课各次实验的题目、代码以及期末复习资料，供大家参考和练习。

## 文件结构

仓库的文件组织结构如下：

- **LAB1 - LAB8**：共有8个实验，每个实验都有对应的文件夹，文件夹内有两个子文件夹：
  - **代码（Code）**：存放MATLAB编写的代码，用于实现实验题目中的相关算法和模型。
  - **题目（Problems）**：包含实验的题目及要求，方便复习和理解实验内容。

- **期末考点（Exam Points.pdf）**：期末考试的重点内容整理，帮助大家集中复习核心知识点。
- **期末复习资料（Review Material.pdf）**：期末复习的综合资料，涵盖理论知识和实际应用的总结。

## 使用说明

1. 浏览各个实验的题目和代码，通过理解实验的具体要求以及MATLAB代码的实现，巩固数学建模的知识和技巧。
2. 使用“期末考点”和“期末复习资料”PDF文档，快速定位复习重点，掌握考试相关内容。
